- 학습 목표 달성 확인 목록
    - [] 시스템 소프트웨어와 애플리케이션 소프트웨어를 설명할 수 있는가?
        : 시스템 소프트웨어 = H/W를 제어하는 S/W (C/C++, Assembly)
          애플리케이션 소프트웨어 = 응용 S/W (메모장, 워드, 포토샵 등)
    - [] 애플리케이션 아키텍처의 종류와 특징을 설명할 수 있는가?
        : Desktop App.(StandAlone App.) = CD게임, C/S = 온라인게임, A/S = 홈페이지, WAS = 네이버 메일, 구글 스타디아
    - [] gradle 빌드 도구를 사용하여 자바 애플리케이션 프로젝트를 구성할 수 있는가?
        : $ gradle init
    - [] gradle을 사용하여 애플리케이션을 실행(run)하고 빌드(build) 할 수 있는가?
        : `$ gradle run`, `$ gradle build`
    - [] gradle을 사용하여 Eclipse IDE 프로젝트로 전환할 수 있는가?
        : build.gradle, `$ gradle eclipse`
    - [] 형상 관리 시스템이 무엇인지 알고 있는가?
        : 소프트웨어 개발 및 유지보수 과정에서 발생하는 소스코드, 문서 등 각종 결과물(형상)에 대한 변경사항을 체계적으로 관리하고 제어하기 위한 활동.
          버전 관리 시스템을 이용.
    - [] CSV, Subversion, Git 버전 관리 시스템의 차이점 이해하고 있는가?
        : CSV = 파일 전체를 서버 저장소에 저장하고 사용 (백업 정보는 서버에만 저장됨)
          Subversion = 파일의 변경된 부분만 서버 저장소에 업로드하고 사용 (위와 동일)
          Git = 로컬 저장소에도 백업 정보를 저장함
    - [] 프로젝트를 git 저장소로 설정할 수 있는가? 
        : `$ git init`, .gitignore
