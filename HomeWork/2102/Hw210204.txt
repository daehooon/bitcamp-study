- 학습 목표 달성 확인 목록
    - [] mutable 객체와 immutable 객체의 차이를 예를 들어 설명할 수 있는가?
        : mutable = 쉽게 변하는 / immutable = 불변
          인스턴스 데이터를 바꿀 수 있음 / 못바꿈
    - [] 리터럴로 만든 String 객체와 new 로 만든 String 객체의 차이점을 설명할 수 있는가?
        : 리터럴로 만든 String 객체는 ` == ` 로 값 비교 가능, new는 equals() 메서드 사용
    - [] 래퍼(wrapper) 클래스의 용도와 사용법을 알고 있는가?
        : primative type data 객체를 모아 한번에 불러올 수 있음
          Object 객체에 담아서 사용
    - [] java.util.Date, java.sql.Date, java.util.Calendar 클래스를 사용할 수 있는가?
    - [] 기능을 확장하기 위해 사용하는 기법의 유형을 설명할 수 있는가?
        : specialization
    - [] 상속을 이용하여 기능을 확장할 수 있는가?
        : extends
    - [] 수퍼 클래스와 서브 클래스의 용어를 이해하는가?
        : class B extends A() {} , A = 수퍼 클래스, B = 서브 클래스
    - [] 상속 관계에 있는 클래스의 인스턴스 생성 과정을 이해하는가?
        : 서브 클래스부터 차근차근 올라가서 최상위 수퍼 클래스의 인스턴스부터 생성됨
    - [] 상속 관계에 있는 클래스의 생성자 호출 과정을 이해하는가?
        : 최상위 수퍼 클래스의 생성자부터 하위 서브 클래스로 차례로 내려가며 호출됨
    - [] 수퍼 클래스의 생성자를 호출하는 `super()` 의 사용법을 아는가?
        : 서브 클래스의 메서드의 가장 첫줄에 호출하여 사용.
          수퍼 클래스의 기본생성자가 없을시에 `super()`에 수퍼클래스가 원하는 타입의 값을 넣어
          수퍼 클래스에 생성자를 만듦
    - [] 자바에서 다중 상속을 지원하지 않는 이유를 아는가?
        : 변수의 타입이 겹칠 수 있어서 혼동됨
    - [] specialization과 generalization 상속 기법을 설명할 수 있는가?
        : 수퍼 클래스에 기능추가 / 교집합된 기능을 모아 수퍼 클래스로 만들어 리펙터링
    - [] 추상 클래스의 용도와 사용법을 아는가?
        : 생성자 호출을 원하지 않을 때, 호출 할 필요 없는 클래스를 명시하기 위해,
          이 클래스(추상 클래스)에 상속된 클래스를 만들어 사용해라.
          class abstract A() {}
