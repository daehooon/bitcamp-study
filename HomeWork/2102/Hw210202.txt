- 학습 목표 달성 확인 목록
    - [] 추상화에 대해 설명할 수 있는가?
        : 사람, 사물, 개념을 클래스로 정의하는 것
    - [] 캡슐화가 무엇인지 설명할 수 있는가?
        : 잘못된 사용을 방지하는 문법
    - [] 캡슐화를 보조하는 접근 제어 문법을 이해하는가?
        : private/(default)/projected/public 
    - [] private/(default)/projected/public 의 접근 범위를 설명할 수 있는가?
        : 같은 클래스/ + 같은 패키지/ + 자식 클래스/ 완전공개
    - [] getter와 setter를 정의하고 사용할 수 있는가?
        : 이클립스 기능 사용 - source - generate Getters and Setters
    - [] 필드(field)와 프로퍼티(property)의 차이점을 아는가?
        : 변수 이름 = 필드, set*** get*** = 프로퍼티
    - [] 상속의 의미와 이점을 이해하는가?
        : 설계도 확장
    - [] 다형성의 3대 유형을 설명할 수 있는가?
        : 다형적 변수, 오버로딩, 오버라이딩
    - [] 다형적 변수를 다룰 수 있는가?
    - [] 오버로딩의 의미와 그 이점을 설명할 수있는가?
        : 프로그래밍의 일관성
    - [] 오버라이딩이 의미와 그 이점을 설명할 수 있는가?
        : 프로그래밍의 일관성
