- 학습 목표 달성 확인 목록
    - [] 메서드를 정의하고 사용할 수 있는가?
        : 명령을 기능단위로 분리
    - [] 파라미터와 아규먼트에 대해 설명할 수 있는가?
        : 파라미터는 값을 받을 변수, 아규먼트는 파라미터에 보내주는 값
    - [] call by value 와 call by reference에 대해 예제를 들어 설명할 수 있는가?
    - [] 클래스 문법의 용도를 설명할 수 있는가?
        : 메서드를 역할(role), 책임(responsibility)에 따라 분류 -> 소스코드의 유지보수가 쉽다
    - [] 역할에 따라 메서드를 분류하여 클래스로 정의할 수 있는가?
        : App.java(메인메서드), MemberHandler.java(멤버 add 및 list, 멤버 변수 배열), Prompt.java(사용자에게 입력받는 명령어, 형변환)
    - [] 클래스에 등록된 메서드를 호출할 수 있는가?
        : Prompt.java ->메서드 호출 App.java -> 호출 명령어 Prompt.**
