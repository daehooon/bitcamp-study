- 학습 목표 달성 확인 목록
    - [] 환경 변수 JAVA_HOME과 PATH를 설정하고 그 용도를 설명할 수 있는가?
        : JAVA_HOME = C:\java\graalvm-**, Java app.과 OS의 연동을 위해
          PATH = C:\java\graalvm-**\bin, javac의 전체경로 제공
    - [] gradle 설정을 통하여 이클립스 프로젝트 이름을 변경할 수 있는가?
        : eclipse {
            project {
            // eclipse .project 파일에 등록할 프로젝트 이름
            name = "프로젝트 이름"
            }
          }
    - [] 메서드를 정의할 수 있는가?
        : 명령어를 기능별로 분리
    - [] 정의한 메서드를 사용할 수 있는가?
    - [] 메서드 시그너처, 메서드 바디, 파라미터, 아규먼트 용어를 설명할 수 있는가?
        : 파라미터 = 값을 받을 변수, 아규먼트 = 파라미터에 넘겨주는 값
    - [] 메서드 문법을 사용했을 때 이점은 무엇인가?
        : 관리 용이(유지보수 용이), 가독성
    - [] 리팩토링의 의미가 무엇인지 아는가?
        - 코드를 묶거나 쪼개서 유지보수 하기 좋은 구조로 소스 코드를 정리하는 것!
