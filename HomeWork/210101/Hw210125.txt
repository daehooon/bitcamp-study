- 학습 목표 달성 확인 목록
    - [] 가비지의 의미를 알고 있는가?
        : 주소를 잃어 사용할 수 없는 메모리
    - [] 레퍼런스 카운트란 무엇인가?
        : 객체의 참조 상태를 관리하기 위해 사용(카운트 증감 -> 카운트가 0이 될시 가비지)
    - [] 가비지 컬렉터는 무엇인지 알고 있는가?
        : 가비지를 메모리에서 해제
    - [] 가비지 컬렉터가 실행되는 경우는 언제인가?
        : 메모리기 부족할 때, CPU가 한가할 때, System.gc() 호출(바로 해제는 안됨)
    - [] System.gc() 메서드는 무슨 일을 하는가?
        : 사용자가 메서드를 호출하여 가비지 컬렉터 실행을 원할 때 바로 실행하지는 않고
          빠른 시간내에 실행 할 것을 요청함.
