- 학습 목표 달성 확인 목록
    - [] 클래스 로딩이 무엇인지 설명할 수 있는가?
        : .class 파일에 들어있는 명령을 실행하기 위해 메모리에 배치하는 것
    - [] 스태틱 필드와 인스턴스 필드의 생성 시점을 이해하는가?
        : 스태틱 필드 = 클래스가 실행될 때
          인스턴스 필드 = new 명령으로 생성
    - [] Method Area와 Heap, Stack 메모리 영역의 쓰임새를 이해하는가?
        : Method Area = 로딩된 클래스(스태틱) 필드 배치, static 필드를 생성
          Heap = 인스턴스 필드 저장
          Stack = 로컬 변수 저장
    - [] 가비지가 무엇인지 설명할 수 있는가?
        : 주소를 잃어버려 사용할 수 없는 인스턴스
    - [] 스태틱 필드의 쓰임새를 아는가?
        : 같은 기능을하는 이름만 다른 클래스를 복제하여 사용 가능
    - [] 인스턴스 필드의 쓰임새를 아는가?
        : 스태틱 필드을 역할 분할해서 인스턴스 필드로 생성시에 복제된 클래스들
          을 줄일 수 있음
    - [] 인스턴스 메서드의 쓰임새를 아는가?
        : 리펙토링
    - [] 인스턴스 메서드의 내장 변수인 this를 설명할 수 있는가?
        : 히든 변수(숨겨진 채 선언되어 있음), 따로 변수선언 없이 인스턴스 메서드 호출 가능
