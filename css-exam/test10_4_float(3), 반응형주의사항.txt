 $ float 3
     => float과 inline-block의 차이

 $ 보통 Grid 형식으로 div 들을 정령할 때 inline-block은 안쓰인다. 여러가지가 얽혀 제어가 힘들기 때문이며 float과 clear를 이용한다. 

 $ 반응형 웹은 보통의 경우 PC를 제외한 스마트 기기 사이에서만 한다.
 $ 네이버나 다음 같은 정보가 많이 출력되는 사이트의 경우 모바일용 페이지가 따로 존재한다.
 $ 많은 정보를 출력 가능한 PC의 장점을 살릴 필요가 없는 경우의 웹사이트만 PC, smart 기기 따로 할 것 없이 같이 반응형으로 만들어도 된다.
