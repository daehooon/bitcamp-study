 $ display 속성 : block, inline, inline-block
    => block
       => 한 줄을 모두 점유한다.
       => 자신의 너비와 높이를 가질 수 있다.
       => 예) h1 ~ h6, div, p, address, ul, ol 등
    => inline
       => 한 줄 속에 포함되어 다른 태그와 그 줄을 함께 공유한다.
       => 자신의 너비와 높이를 가질 수 없다.
          콘텐츠의 크기에 따라 너비와 높이가 결정된다.
       => a, i, b, strong, span 등
    => inline-block
       => inline 처럼 한 줄을 다른 태그와 공유한다.
       => block 처럼 자신의 너비와 높이를 가질 수 있다.
